### TaTi
> Copilote IA open source orienté delivery & ops, connecté via MCP

- Connecte vos outils réels : Slack, PostgreSQL, GitHub, Grafana, Airflow, dbt, AWS, GCP, Azure…
- Orchestration de **+20 connecteurs MCP** (messagerie, bases de données, cloud, observabilité)
- Déploiement simple via **Docker Compose**
- Auth locale avec sessions configurables
- Disponible en 🇫🇷 Français, 🇬🇧 English, 🇨🇳 中文

- Interface dédiée chauffeur & passager
- Notation et feedback post-course
